## 1.2 배열의 생성
### 1. 리스트에서 행렬/배열 생성

```python
import numpy as np
a = [[1, 2, 3], [4, 5, 6]]
b = np.array(a)
print(b)

```
결과:
[[1 2 3]
 [4 5 6]]


``` 
print(b.ndim)  # 배열의 열수(차원)

```
결과:2
```
print(b[0,0])  # 배열의 원소 접근
```
결과:1
```
print(b.shape) # 배열의 차원
```
결과:(2, 3)
